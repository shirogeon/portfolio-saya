# Leon Web Developer Portfolio

Website statis yang direalisasikan dari desain ZIP: layout portfolio minimal, grid background, dark technical cards, responsive navbar, project modal, skill progress bars, dan contact section.

## Cara pakai
1. Buka `index.html` langsung di browser, atau upload semua isi folder ini ke hosting.
2. Edit teks utama di `index.html`.
3. Edit tampilan di `style.css`.
4. Edit data popup project di `script.js`.

## Struktur
```
index.html
assets/
  css/style.css
  js/script.js
  img/hero-code.svg
  img/project-cafe.svg
  img/project-system.svg
```
